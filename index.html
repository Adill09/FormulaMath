ЕНТ Формулалары по математике

Туынды
Норм тема
  1. \(\frac{d}{dx}(c) = 0 \quad (c = \text{константа})\)
  2. \(\frac{d}{dx}(x) = 1\)
  3. \(\frac{d}{dx}(x^n) = n x^{n-1}\)
  4. \(\frac{d}{dx}(a^x) = a^x \ln a\)
  5. \(\frac{d}{dx}(e^x) = e^x\)
  6. \(\frac{d}{dx}(\ln x) = \frac{1}{x}\)
  7. \(\frac{d}{dx}(\sin x) = \cos x\)
  8. \(\frac{d}{dx}(\cos x) = -\sin x\)
  9. \(\frac{d}{dx}(\tan x) = \frac{1}{\cos^2 x}\)
  10. \(\frac{d}{dx}(u \pm v) = u' \pm v'\)
  11. \(\frac{d}{dx}(u \cdot v) = u'v + uv'\)
  12. \(\frac{d}{dx}\!\Big(\frac{u}{v}\Big) = \frac{u'v - uv'}{v^2}\)
Интергал
Қазіргі тақырып
  1. \(\int 0 \, dx = C\)
  2. \(\int k \, dx = kx + C \quad (k = \text{константа})\)
  3. \(\int x^n dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)\)
  4. \(\int \frac{dx}{x} = \ln|x| + C\)
  5. \(\int e^x dx = e^x + C\)
  6. \(\int a^x dx = \frac{a^x}{\ln a} + C\)
  7. \(\int \sin x dx = -\cos x + C\)
  8. \(\int \cos x dx = \sin x + C\)
  9. \(\int \frac{dx}{\cos^2 x} = \tan x + C\)
  10. \(\int \frac{dx}{\sin^2 x} = -\cot x + C\)
Тригонометрия
хз 9 классқо
  1. \(\sin(x \pm y) = \sin x \cos y \pm \cos x \sin y\)
  2. \(\cos(x \pm y) = \cos x \cos y \mp \sin x \sin y\)
  3. \(tg(x \pm y) = \frac{tg x \pm tg y}{1 \mp tg x \cdot tg y}\)
  4. \(\cos x \cdot \cos y = \tfrac{1}{2} \big( \cos(x-y) + \cos(x+y) \big)\)
  5. \(\sin x \cdot \sin y = \tfrac{1}{2} \big( \cos(x-y) - \cos(x+y) \big)\)
  6. \(\sin x \cdot \cos y = \tfrac{1}{2} \big( \sin(x+y) + \sin(x-y) \big)\)
  7. \(\cos x + \cos y = 2 \cos \tfrac{x+y}{2} \cdot \cos \tfrac{x-y}{2}\)
  8. \(\cos x - \cos y = -2 \sin \tfrac{x+y}{2} \cdot \sin \tfrac{x-y}{2}\)
  9. \(\sin x \pm \sin y = 2 \sin \tfrac{x \pm y}{2} \cdot \cos \tfrac{x \mp y}{2}\)
  10. \(\sin 2x = 2 \sin x \cos x\)
  11. \(\cos 2x = \cos^2 x - \sin^2 x\)
  12. \(\cos 2x = 2 \cos^2 x - 1\)
  13. \(\cos 2x = 1 - 2 \sin^2 x\)
  14. \(\cos^2 x = \tfrac{1 + \cos 2x}{2}\)
  15. \(\sin^2 x = \tfrac{1 - \cos 2x}{2}\)
  16. \(tg 2x = \tfrac{2 tg x}{1 - tg^2 x}\)
